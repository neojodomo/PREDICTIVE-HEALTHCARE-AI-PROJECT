## Compressed Files

The `compressed_files/` folder contains ZIP archives with:
- Prediction code (`src/`)
- Trained models (`models/`)
- Input schemas (`schemas/`)

## How to Use

1. Download the ZIP file from `compressed_files/`
2. Extract the contents
3. Use the extracted `src/`, `models/`, and `schemas/` folders
   for backend integration

## Notes

- The ZIP files must be extracted before use
- Feature names must match training features exactly
- This repository is for inference only
